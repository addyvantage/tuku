# Security Policy

## Supported Versions

Tuku currently supports security fixes on the actively developed branch:

| Version | Supported |
| --- | --- |
| `main` | Yes |
| older commits/tags | No |

## Reporting a Vulnerability

Please report vulnerabilities privately through GitHub Security Advisories for this repository.

- Preferred: GitHub "Report a vulnerability" (private disclosure).
- Include: reproduction steps, impact, affected paths, and environment details.
- Avoid public disclosure until a maintainer acknowledges and coordinates a fix.

## Response Expectations

- Initial acknowledgment target: within 7 days.
- Mitigation/fix timeline depends on severity and exploitability.
- We may request additional reproduction details before issuing a patch.

Thank you for helping keep Tuku safe and trustworthy.
